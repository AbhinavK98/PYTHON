# Interview Templates

## What is it?

Copy-ready code skeletons for common patterns.

## Why do we need it?

Under pressure, templates prevent syntax errors and forgotten edge cases.

## Real-world analogy

Pilot checklist before takeoff — same every flight.

## Visualization

```
Template -> customize -> dry run -> submit
```

## Where interviewers ask it

Live coding rounds.

## Advantages

- Speed
- Consistency

## Disadvantages

- Must still adapt; blind paste fails follow-ups

## When NOT to use it

When problem is pure simulation with no standard pattern.

## Interview Tricks

- Practice writing from memory weekly
