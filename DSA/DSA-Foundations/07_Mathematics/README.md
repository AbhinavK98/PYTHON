# Mathematics for DSA

## What is it?

Number theory, primes, GCD, modular arithmetic used in interviews.

## Why do we need it?

Many problems reduce to math (divisibility, combinatorics, sieve).

## Real-world analogy

GCD = largest tile size that fits two rectangles without cutting.

## Visualization

```
gcd(12,18)=6
```

## Where interviewers ask it

Prime problems, modulo, combinations, geometry basics.

## Advantages

- Can reduce O(n) to O(sqrt n) or O(log n)

## Disadvantages

- Easy to make off-by-one errors in formulas

## When NOT to use it

When simulation is simpler than formula derivation under time pressure.

## Interview Tricks

- Memorize gcd, sieve templates
