# DSA Learning Repository

This repository has two tracks:

1. **[DSA Foundations](DSA-Foundations/README.md)** — thinking, complexity, recursion, searching, sorting (start here)
2. **[Arrays Patterns](Arrays/README.md)** — LeetCode-style pattern practice (after foundations)

---

## Learning Roadmap

### Phase 0: Foundations (required first)

Complete [DSA-Foundations](DSA-Foundations/README.md) before heavy LeetCode grinding.

### Phase 1: Arrays Patterns

1. Start with `01_HashMap_Complement` to build lookup intuition.
2. Move to `02_Two_Pointers` for in-place and pair-based strategies.
3. Learn `03_Prefix_Sum` to answer range queries fast.
4. Practice `04_Sliding_Window` for contiguous subarray optimization.
5. Master `05_Kadanes` for best subarray optimization.
6. Study `06_Greedy_Array` for local-decision global-optimum patterns.
7. Finish with `07_Matrix` for 2D array transformations.

## Recommended Order

- HashMap Complement -> Two Pointers -> Prefix Sum -> Sliding Window -> Kadane's -> Greedy Arrays -> Matrix
